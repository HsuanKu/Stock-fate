# Plum Blossom Stock Divination (梅花易數股市預測) v2.0

This project combines traditional Chinese Metaphysics (Meihua Yishu) with Google Gemini AI to predict stock market trends.

## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Vercel Serverless Functions
- AI: Google Gemini API (@google/genai)

## Deployment
Deployed on Vercel.
